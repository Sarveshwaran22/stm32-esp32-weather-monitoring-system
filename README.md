# STM32-ESP32 Weather Monitoring System

## Project Overview
This project is a real-time weather monitoring system developed using STM32 and ESP32 microcontrollers. The system monitors environmental parameters and enables wireless data communication for remote monitoring applications.

## Features
- Real-time weather monitoring
- Temperature sensing
- Humidity monitoring
- Wireless communication using ESP32
- Embedded system design using STM32
- Low-power operation

## Hardware Components
- STM32 Microcontroller
- ESP32 Wi-Fi Module
- Temperature and Humidity Sensor
- Power Supply
- Connecting Wires

## Software Tools
- Arduino IDE
- Embedded C/C++
- STM32 Board Package
- ESP32 Board Package

## Working Principle
The sensor continuously collects environmental data. STM32 processes the sensor readings and ESP32 provides wireless communication for transmitting the collected information. The system can be used for weather monitoring and IoT-based applications.

## Applications
- Smart Agriculture
- Weather Stations
- Environmental Monitoring
- IoT Projects
